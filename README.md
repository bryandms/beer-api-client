## Getting Started

### Installation

**Step 1:** Clone or download the [repo](https://github.com/bryandms/beer-api-client).

<br />

**Step 2:** Switch to your repository's directory.

```bash
cd /path/to/your/repo
```

<br />

**Step 3:** Install dependencies.

```bash
npm install
```

<br />

**Step 4:** Run the app.

```bash
npm run start
```
